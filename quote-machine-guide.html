TRVE Quote Machine AI

AI-powered endpoint reference — Safari Operations Platform

Overview

The Quote Machine is a suite of seven AI-powered API endpoints that automate the most time-consuming parts of a safari enquiry workflow — from parsing raw email text all the way through to writing a polished, personalised quotation narrative.

All endpoints require a valid Authorization: Bearer <token> header and return application/json unless otherwise noted.

Endpoints at a Glance

POST/intake/parse

Parses raw enquiry text (email, WhatsApp, form) into a structured JSON record and creates an enquiry in the database.

GET/advisory

Returns season, weather, and permit advisories for a destination + date range. Responses cached for 24 hours.

POST/itinerary/generate

Generates a complete day-by-day itinerary from a client brief including destinations, lodge names, and duration.

POST/itinerary/regenerate-day

Rewrites a single day's guest-facing description without touching the rest of the itinerary.

POST/pricing/calculate

Calculates itemised trip pricing with child banding, seasonal rates, and configurable markup.

POST/quote/narrative

Writes a professional quotation narrative: opening, lodge highlights, investment note, and call-to-action.

POST/quote/upsells

Suggests three geographically-relevant, budget-appropriate add-ons to include in the quote.

Query Parameters — GET /advisory

ParameterTypeRequiredDescription
destinationstringYesPark, region, or country name
startYYYY-MM-DDYesFirst day of travel
endYYYY-MM-DDYesLast day of travel
refreshbooleanNoPass true to bypass the 24h cache

Caching

Advisory responses are stored in the ai_cache database table and reused for 24 hours. If the underlying data changes (e.g. a permit policy update), append ?refresh=true to force a fresh AI call.

The ai_cache table is created automatically on server startup via _ensure_ai_cache_table() — no manual migration required.
